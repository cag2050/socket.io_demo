### socket.io 库的使用

说明：
1. 因为本项目socket.io使用的新版本，跟书上旧版本的socket.io使用写法有差异
2. send对应on('message', fn)，emit对应on
3. 使用命名空间时，js里是：`on('connect', fn)`
